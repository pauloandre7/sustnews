<h2>Descrição do Projeto</h2>
<p>
  Projeto em desenvolvimento para disciplina Desenvolvimento Web Front-End do curso de Engenharia de Software da Universidade Tecnológica Federal do Paraná. O Sustnews
  é um site para agregar notícias sustentáveis do paraná e compartilhar ações comunitárias/sustentáveis da cidade de cornélio procópio, facilitando o acesso à informações
  sobre políticas de sustentabilidade e oportunidades de voluntariado.
</p>

<h2>Acesso ao Projeto</h2>
<p>
  ❗ O projeto ainda se encontra em desenvolvimento e sem nenhuma versão hospedada em servidor online.
</p>
